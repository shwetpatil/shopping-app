Key Features:
=============
✅ 6 Independent Microfrontends (each deployable separately)
✅ 8 Backend Microservices (event-driven with Kafka)
✅ Complete Docker setup for all services
✅ Comprehensive documentation
✅ Team-based organization (Platform, Search, Engagement, Commerce)
✅ Modern tech stack (Next.js 14, React 18, TypeScript, TanStack Query)
✅ Production-ready configurations
✅ B2B multi-tenant support

Port Allocation:
================
Frontend:
- 3000: Shell App (Host)
- 3001: Search MFE
- 3002: Wishlist MFE
- 3003: Reviews MFE
- 3004: Products MFE
- 3005: Cart MFE

Backend:
- 8080: API Gateway
- 3010: Auth Service
- 3011: Product Service
- 3012: Order Service
- 3013: Cart Service
- 3014: Payment Service
- 3015: Inventory Service
- 3016: Notification Service

Infrastructure:
- 5432: PostgreSQL (multiple databases)
- 6379: Redis
- 9092: Kafka
- 2181: Zookeeper
